# Basic Calculator

A clean, menu‑driven command‑line calculator built in Python. It performs basic arithmetic operations with proper input validation and user guidance.

## Features
- Addition  
- Subtraction  
- Multiplication  
- Division (with division‑by‑zero handling)  
- Continuous menu workflow  
- Numeric input validation  
- Clear result output  

### Clone the repository

```bash
git clone https://github.com/<your-username>/basic-calculator.git
cd basic-calculator
```

## How to Run
```bash
python Basic_Calculator.py
```

## Usage
Select an operation (1–5), enter two numbers, and view the result. The menu repeats until you exit.
